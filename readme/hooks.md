类组件的不足

状态逻辑复用难
  1.缺少复用机制
  2.渲染属性和高阶组件导致层级冗余

趋向复杂难以维护
  1.生命周期函数混杂不相干逻辑
  2.相关逻辑分散在不同生命周期

this指向困扰
  1.内联函数过度创建新句柄
  2.类成员函数不能保证this

函数组件无this问题；
自定义Hook方便复用状态逻辑；
